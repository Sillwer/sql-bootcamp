# Портфолио SQL проектов

## Обзор
11 учебных проектов по SQL, охватывающих основные аспекты работы с базами данных.

## Основные темы

### Базовые навыки
- SELECT-запросы, фильтрация, сортировка
- JOIN операции, работа с множествами
- Изменение данных (INSERT, UPDATE, DELETE)

### Продвинутые техники
- Представления и материализованные представления
- Индексы и оптимизация запросов
- Агрегация и аналитические функции

### Архитектура БД
- Транзакции и уровни изоляции
- Функции и триггеры
- Ограничения целостности данных

## Технологии
- PostgreSQL
- DDL/DML операции
- Рекурсивные запросы
- Оптимизация производительности

Проекты демонстрируют постепенное освоение SQL от базовых конструкций до решения практических задач работы с данными.
